📊 ShopOS – Shop Management Dashboard

A modern and responsive Shop Management System Dashboard built using HTML, CSS, and JavaScript. This project provides an intuitive interface for managing daily shop operations with a focus on simplicity, usability, and clean UI design.

🚀 Features
📈 Dashboard Overview
Displays key metrics like daily revenue, orders, low stock items, and total customers
🧾 Recent Transactions
View detailed transaction history with customer name, items, amount, and status
⚠️ Low Stock Alerts
Highlights products that need restocking
🧭 Sidebar Navigation
Easy access to Inventory, Sales, Customers, Suppliers, and Reports
🔍 Search Functionality
Quick search bar for efficient navigation
🌙 Dark UI Design
Modern dark-themed interface for better user experience
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & layout (dark theme UI)
JavaScript (Vanilla JS) – Interactivity
